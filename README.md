# Tecnologias para Inserção Digital - Bairro Vista Alegre

## 📍 Local do Projeto
Associação de Moradores do Bairro Vista Alegre – Videira/SC

## 🎯 Objetivo
Promover a inclusão digital da comunidade por meio da capacitação no uso de tecnologias básicas como internet, e-mail e pacote Microsoft Office, além da introdução ao HTML.

## 💻 Tecnologias Utilizadas
- HTML
- Microsoft Word
- Microsoft Excel
- Microsoft PowerPoint
- Gmail
- Google Chrome

## 📚 Materiais Desenvolvidos
- Apostila digital
- Slides das aulas
- Exercícios práticos
- Página simples em HTML
- Cartilha de segurança digital

## 👥 Público Atendido
12 participantes da comunidade

## 📊 Resultados
- 90% relataram melhora no uso de tecnologias
- 85% concluíram o curso
- 70% utilizaram materiais fora das aulas
- Redução de 80% na insegurança digital
